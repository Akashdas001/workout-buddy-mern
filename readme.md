## BACKEND :

npm install nodemon    (if we anything changes of our code we donot need to again and again to kill the server and start that , nodemon handel that problem)

npm run dev            (for running the server)

npm install dotenv ( install env file)

npm install mongoose

## API Endpoints:

GET    /workouts --> Get all the Workout docs
POST    /workouts --> Create a new Workout doc
GET    /workouts/:id --> Get a single workout doc
DELETE /workouts/:id --> Deletes a single workout doc
PATCH /workouts/:id --> Updates a single workout
